Installation Steps

1

Clone the Repository

git clone https://github.com/your-username/ai-college-assistant.git
2

Install Dependencies

pip install streamlit langchain faiss-cpu chromadb openai python-dotenv PyPDF2
3

Set up Environment Variables

Create a .env file with your OpenAI API key:

OPENAI_API_KEY=your_api_key_here
4

Run the Application

streamlit run app.py
